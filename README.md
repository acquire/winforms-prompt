# winforms-prompt
A winforms user input dialog inspired by Javascript window.Prompt().

#usage

```c#
Prompt.ShowDialog();
```

![alt tag](prompt1.png)

```c#
Prompt.ShowDialog("Apple Count", "How many apples do you have?");
```

![alt tag](prompt2.png)

```c#
Prompt.ShowDialog("Apple Count", "How many apples do you have?", "0");
```
